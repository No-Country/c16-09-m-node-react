### Backend

### Instalar paquetes

```shell
npm i
```

### Configurar archivo .env

```js
DB_USER=
DB_PASS=
DB_NAME=
```

### Setup sequelize

Agregar base de datos (usando migration)

```shell
npx sequelize-cli db:migrate
```

### Start the app

Igresa a la carpeta /src y ejecuta el comando

```shell
/backend/src
```

```shell
npx nodemon app
```
